== Description ==

Adds a WooCommerce debug panel to the Debug Bar plugin. Requires the [Debug Bar plugin](http://wordpress.org/extend/plugins/debug-bar/) to function.

== Installation ==

1. Upload `woocommerce-debug-bar` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Access the 'WooCommerce' tab of the 'Debug' menu in the Toolbar from any page
